Overview

A filter which returns documents which fall into a specified geographical distance range

Examples:

Define the filter with a hash


search do
  query do
    filtered do
      filter do
        geo_distance location: '50.090223,14.399590', gte: '2km', lte: '5km'
      end
    end
  end
end
