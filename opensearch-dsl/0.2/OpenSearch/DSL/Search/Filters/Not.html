Since â€˜not` is a keyword in Ruby, use the `_not` method in DSL definitions

A filter which takes out documents matching a filter from the results

Examples:

Pass the filter as a Hash

search do
  query do
    filtered do
      filter do
        _not term: { color: 'red' }
      end
    end
  end
end

Define the filter with a block


search do
  query do
    filtered do
      filter do
        _not do
          term color: 'red'
        end
      end
    end
  end
end
