Overview

A query which returns documents which fall into a specified geographical shape

Examples:


search do
  query do
    geo_shape :location do
      shape type: 'envelope',
            coordinates: [[14.2162566185,49.9415476869], [14.7149200439,50.1815123678]]
    end
  end
end
